  17.  trait Tag[T] extends AnyRef

    Like scala.reflect.api.TypeTags#TypeTag, but supports higher-kinded type tags via TagK type class.

    Like scala.reflect.api.TypeTags#TypeTag, but supports higher-kinded type tags via TagK type class.

    In context of DI this allows you to define a module parameterized by a generic higher-kinded type parameter. This is especially helpful when coding in a tagless final style

    Example:

    class MyModule[F[_]: Monad: TagK] { make[MyService[F]] make[F[Unit]].named("empty").from(Monad[F].pure()) }

    Without a TagK constraint above, this example would fail with no TypeTag available for MyService[F]
